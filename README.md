# House Price Prediction – Advanced Machine Learning

## Project Overview

This project focuses on predicting residential house prices using advanced Machine Learning techniques. The workflow includes extensive data preprocessing, exploratory data analysis (EDA), feature engineering, missing value treatment, feature encoding, model building, hyperparameter tuning, and performance evaluation. Multiple regression models were compared to identify the most accurate model for house price prediction.

##  Problem Statement

The objective of this project is to build a predictive model that estimates the selling price of a house based on its physical characteristics, location, and other property-related features.

## Dataset

The dataset contains various attributes related to residential properties, including:

- Lot Area
- Overall Quality
- Year Built
- Overall Condition
- Basement Area
- Garage Area
- Number of Bedrooms
- Number of Bathrooms
- Neighborhood
- Kitchen Quality
- Exterior Features
- Sale Price (Target Variable)
